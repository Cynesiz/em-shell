Hints

  • "ls" doesn't work, but tab-completion does.
  • Try "cd <tab>" to see the root directory contents
  • Up and Down arrows move through history
  • Scrollback: use mouse wheel or Shift+PgUp, Shift+PgDn
  • If you leave it idle for a couple minutes then a request will time out and it will hang
  • Implementation details and source are on GitHub

Examples

echo "test file" > myfile
read var < myfile
echo $var
